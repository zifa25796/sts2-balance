# Notes

- Numbers shown as `A (B) → C (D)` represent normal/enraged/alternate values where applicable.
- Changes are focused on:
  - Reducing unavoidable early damage spikes.
  - Lowering excessive multi-action enemy pressure.
  - Improving underperforming relics and events.
  - Maintaining unique enemy mechanics through redistribution rather than simple nerfs.


## Requirements

- Slay the Spire 2 (v0.108.0+)
- [BaseLib](https://github.com/STS2Modding/BaseLib) v3.3.2+

## Installation

1. Install [BaseLib](https://github.com/STS2Modding/BaseLib) mod
2. Place the `balance` folder into your `STS2_Data/Mods/` directory
3. Enable the mod in the in-game mod manager

## Build

```shell
dotnet build -c ExportRelease
```

The built mod will be output to `bin/ExportRelease/net9.0/`.
